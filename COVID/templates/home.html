AUEB NLP Group COVID-19 Search Engine

An Experimental Document and Snippet Retrieval Search Engine for the COVID-19 Open Research Dataset (CORD-19) based on award winning technology developed by AUEB's NLP Group for the BioASQ challenge

Technology ported to CORD-19 by Dimitris Pappas (email: pappasd@aueb.gr) and Petros Stavropoulos (email: pstav1993@aueb.gr).

Papers:

1: Deep Relevance Ranking Using Enhanced Document-Query Interactions
2: AUEB at BioASQ 7: Document and Snippet Retrieval

Code: aueb-bioasq7

API example: curl --header "Content-Type: application/json" --request POST --data '{"question" : "What do we know about Covid-19 vaccines ?", "section" : ""}' http://cslab241.cs.aueb.gr:5000/just_the_json

All provided for research and demo purposes only. Search engine running on a weak server. Please use responsibly.

If you can host the search engine on a stronger server, or if you can help improve the UI of the web demo, please let us know.
